## Мини‑игра «Поймай штраф»

Автономная браузерная мини‑игра в одном файле `index.html` (HTML + CSS + JavaScript), без внешних библиотек.  
Работает локально и готова к публикации на GitHub Pages.

### Как запустить локально

- Откройте файл `index.html` в браузере (двойной клик).

### Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub (например, `poimai-shtraf`).
2. Загрузите в корень репозитория файлы:
   - `index.html`
   - `README.md`
3. В GitHub откройте: **Settings → Pages**.
4. В разделе **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (или `master`)
   - **Folder**: `/ (root)`
5. Нажмите **Save**.
6. Подождите 1–2 минуты — GitHub покажет ссылку на сайт в этом же разделе Pages.

### Если игра не обновилась сразу

- Подождите немного (иногда GitHub Pages обновляется с задержкой).
- Сделайте жёсткое обновление страницы в браузере (Ctrl+F5 / Ctrl+Shift+R).
